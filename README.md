# Artline Portfolio

## 📌 Sobre o Projeto

O Artline é um sistema web desenvolvido para auxiliar empresas do segmento de molduras, quadros, impressões e decoração na elaboração de orçamentos personalizados.

O projeto foi concebido para centralizar cálculos, gerenciamento de clientes, autenticação de usuários e controle administrativo em uma única plataforma, reduzindo erros operacionais e agilizando o processo comercial.

---

## 🎯 Objetivo

Desenvolver uma plataforma capaz de:

- Gerenciar clientes e usuários;
- Controlar permissões de acesso;
- Automatizar cálculos de orçamento;
- Calcular molduras, vidros, passe-partout, fundos e impressões;
- Integrar autenticação segura;
- Fornecer uma interface administrativa para gerenciamento do sistema;
- Centralizar regras de negócio do segmento de emolduramento.

## 🏗 Principais Funcionalidades

### Autenticação

- Login de usuários
- Recuperação de senha
- Controle de acesso por perfil

### Administração

- Cadastro de clientes
- Controle de usuários
- Ativação e desativação de contas
- Gestão administrativa

### Orçamentos

- Cálculo automatizado de molduras
- Cálculo de vidros
- Cálculo de passe-partout
- Cálculo de fundos
- Cálculo de impressões
- Cálculo de reforços estruturais
- Controle de categorias diversas

### Segurança

- Row Level Security (RLS)
- Controle de permissões
- Proteção de dados
